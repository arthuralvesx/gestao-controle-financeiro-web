# Apresentação do projeto FFB

Esta pasta contém os arquivos de apresentação do Projeto Integrador SENAC.

- `output/apresentacao-ffb-google-slides.pptx`: versão recomendada para importar no Google Slides.
- `output/apresentacao-ffb-controle-financeiro.pdf`: versão para leitura e compartilhamento.
- `output/apresentacao-ffb-controle-financeiro.html`: versão para apresentar pelo navegador.

Para usar no Google Slides, envie o arquivo `.pptx` para o Google Drive e abra com o Google Apresentações.
